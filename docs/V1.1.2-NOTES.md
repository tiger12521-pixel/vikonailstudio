# Vico Nail Story v1.1.2

## Release Goal

Upgrade the latest-promotion section from one active campaign to a reusable multi-campaign carousel while preserving every other v1.1.1 website section.

## Added

- Added a permanent loyalty reward campaign.
- Added desktop and mobile campaign poster assets.
- Added multiple active promotion support.
- Added automatic carousel playback.
- Added previous and next buttons.
- Added slide indicator dots.
- Added touch and pointer swipe navigation.
- Added pause on hover and keyboard focus.
- Added reduced-motion handling.
- Added permanent campaign support through `isPermanent`.
- Added pinned campaign ordering through `isPinned`.
- Preserved date-based visibility, active switches, sorting, poster enlargement, and fallback content.

## Promotion Ordering

1. Promotions with `isPinned: true`.
2. Remaining promotions ordered by `sortOrder`.

## Permanent Campaign

A promotion with `isPermanent: true` does not require an end date. It remains visible while `isActive` is true and its optional start date has been reached.

## Future Upgrade

The JSON data format remains compatible with the planned v1.2.0 administration system. The static data service can later be replaced by a Cloudflare API without rebuilding the carousel component.
