# V1.3.1 SEO 基礎優化

## 新增

- 根目錄 `robots.txt`，允許公開頁面並排除 Admin 與管理 API。
- 根目錄 `sitemap.xml`，收錄首頁與 Gallery 公開頁面。
- `site.webmanifest` 與網站圖示。
- 首頁與 Gallery 的 canonical、description、Open Graph、Twitter Card。
- 首頁 `BeautySalon` JSON-LD 結構化資料。
- 首頁品牌標題調整為語意正確的 `h1`。

## 注意

- Google Search Console 網域驗證建議使用 Cloudflare DNS TXT，不需要把驗證碼寫進 Repository。
- 店家完整街道地址與營業時間尚未寫入結構化資料，避免在未確認資料前自行推測。
