# V1.0.0 架構收尾

此版本以 Stage 3 測試通過的網站為基礎，未改動使用者可見內容與預約功能。

- 根目錄 `README.md`

- `.gitattributes`：統一 LF，圖片以 binary 處理
- `docs/ARCHITECTURE.md`
- `docs/CODING_STYLE.md`
- `docs/RELEASE.md`
- `docs/CHANGELOG.md`

- 完整化 `scripts/update-version.js`

## 版本工具

```bash
npm run version:update -- v1.0.0
npm run version:check
```

`version:update` 會同步更新：

- `version.json`
- `package.json`
- `package-lock.json`

並寫入目前 Git Commit、Branch 與建置時間。

## 正式發布前

`version.json` 的 Commit 預設為 `pending-release`，是刻意保留的安全提示。將此版本複製進正式 Git 專案並完成主要程式 commit 後，必須執行：

```bash
npm run version:update -- v1.0.0
npm run version:check
```

確認通過後再提交 metadata、建立 `v1.0.0` Tag 並 Push。
