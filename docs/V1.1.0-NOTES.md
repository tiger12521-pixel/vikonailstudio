# V1.1.0 開發說明

## 本版目標

V1.1.0 將網站從作品與價目展示，延伸為可在預約前提供完整資訊的內容型網站。

## 新增區塊

### 最新優惠

- 放置於 Hero 後方。
- 目前先保留版面骨架與社群連結。
- 尚未決定使用單張海報、輪播或資料驅動內容。
- 後續討論完成後，只需替換此區，不影響其他頁面。

### 預約須知

- 整理預約時需要提供的項目。
- 說明款式照片、卸甲、延甲與指甲近照需求。
- 加入預約確認、訊息回覆、取消、改期與遲到規則。
- 文字保留原本服務流程的意圖，但改為較適合網站閱讀的表達。

### 美甲常見問題

- 使用原生 `details` 與 `summary` 製作 Accordion。
- 不依賴 JavaScript，即使腳本未載入仍可展開閱讀。
- 包含凝膠維持、脫落原因、術後保養、卸甲與異常甲況等內容。
- 對醫療相關內容採保守表述，並提醒異常時優先尋求專業醫療評估。
