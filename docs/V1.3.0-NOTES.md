# V1.3.0 Gallery System

- Built directly on V1.2.0.
- Homepage now loads the latest six featured works.
- Added `/gallery/` with multi-category filters and lightbox navigation.
- Added Gallery Management tab to `/admin/`.
- Local development stores metadata in `data/gallery.json` and images in `assets/uploads/gallery/`.
- Cloudflare deployment stores both JSON metadata and gallery images in the `GALLERY_ASSETS` R2 bucket.
- Data model includes `artistId`, `sortOrder`, `isActive`, timestamps, and category arrays for future expansion.
