# V1.0.1 發布說明

## 版本定位

`v1.0.1` 是維護與發布流程修正版，不改變網站畫面、預約功能或 API 行為。

## 本版內容

- 新增 `scripts/release.js`，集中處理版本 metadata、檢查、Commit、Tag 與 Push。
- 新增 `npm run release -- <版本>` 指令。
- 發布前自動確認工作目錄乾淨、Branch 為 `main`、測試通過且 Tag 不重複。
- `--commit` 可自動建立 metadata Commit 與 annotated Tag。
- `--push` 可在 `--commit` 基礎上推送 `main` 與 Tag。
- 更新發布文件、常用指令與問題排除說明。

## 不受影響項目

- 網站 HTML 內容。
- CSS 外觀與響應式排版。
- Booking 前端模組。
- Booking API 與 Google Calendar 整合。
- Console 顯示格式。

## 建議發布方式

完成本機測試並先提交本版程式後，執行：

```bash
npm run release -- v1.0.1 --commit --push
```

腳本會將「執行 Release 指令前的 HEAD」寫入 `version.json` 的 `commit` 欄位，作為本次主要程式內容的來源 Commit。
