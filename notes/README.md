# 開發知識庫索引

這個資料夾用來保存「幾個月或幾年後重新維護時，最容易忘記的事情」。

建議閱讀順序：

1. [`QUICK_START.md`](QUICK_START.md)：先把網站跑起來。
2. [`PROJECT_CONTEXT.md`](PROJECT_CONTEXT.md)：快速了解網站用途與目前狀態。
3. [`COMMON_COMMANDS.md`](COMMON_COMMANDS.md)：查 Git、npm、版本與發布指令。
4. [`AI_GUIDE.md`](AI_GUIDE.md)：交給 ChatGPT 或其他協作者閱讀。
5. [`DEPLOY_CHECKLIST.md`](DEPLOY_CHECKLIST.md)：正式上線前逐項確認。
6. [`TROUBLESHOOTING.md`](TROUBLESHOOTING.md)：遇到錯誤時先查這裡。

其他紀錄：

- [`PROJECT_DECISIONS.md`](PROJECT_DECISIONS.md)：記錄重要技術決策與原因。
- [`PROJECT_PHILOSOPHY.md`](PROJECT_PHILOSOPHY.md)：專案長期維護原則。
- [`PROMPTS.md`](PROMPTS.md)：常用 ChatGPT 指令範本。
- [`TODO.md`](TODO.md)：未來功能規劃。
- [`LEARNING_LOG.md`](LEARNING_LOG.md)：開發過程學習紀錄。
